<h1>🚀 DivinityEconomy 3.5.1 (1.21) 🚀</h1>
<h2>➕ Updates ➕</h2>
<ul>
<li>
    <h4>Critical Bug Fix</h4>
    <p>Fixed a critical bug that caused the plugin to crash when a player attempted to value an item (with an extremely large amount)</p>
</li>
<li>
    <h4>Improved Placeholders</h4>
    <p>Placeholders relating to items, such as their value, stock, etc, now support _ (underscores) in the placeholder. This should mean that items with underscored names, such as 'GRASS_BLOCK' will be successfully valued by these commands.</p>
</li>
</ul>
<h2>🪲 Bug Fixes 🪲</h2>
<ul>
<li>Fixed the /setValue command not correctly setting the stock of an item to represent the value.</li>
<li>Fixed the command completion for /setvalue and /setstock not working correctly.</li>
</ul>
